Proyectos Destacados

Tienda en línea para gimnasio

Diseñé y desarrollé una tienda virtual para un gimnasio local llamada EVOFIT, donde se ofrecen suplementos, membresías y productos deportivos. Implementé el diseño responsivo con Bootstrap, pasarela de pago y un sistema de control básico de inventario en PHP y MySQL.

Tecnologías: PHP, Bootstrap, MySQL, HTML, CSS

CRM para gestión de prospectos

Este proyecto es uno de los más importantes que he desarrollado. Se trata de un CRM personalizado con tablero Kanban para seguimiento de prospectos, con funcionalidades de arrastrar y soltar, filtro por fechas, edición dinámica y conexión a una API RESTful.

Utiliza PHP con Slim Framework en el backend, jQuery y AJAX en el frontend, y está pensado para escalar con facilidad.

Tecnologías: PHP (Slim), MySQL, JavaScript, jQuery, Bootstrap, AJAX
